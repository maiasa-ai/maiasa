# Maiasa

My new favorite AI.

☀️ [maiasa.ai](https://www.maiasa.ai)

*Maiasa* is a next-gen, open-source language model parody that's lightning-fast, robust, safe, ethical and sustainable. You can try it for free right now!

Benefits

- ☀️ **Fast**: Often generates up to 1,000 tokens per second.
- ☀️ **Robust**: Under the hood, it's always easy to understand why the model said what it said.
- ☀️ **Efficient**: Runs even on low-end machines, and takes very little disk space.
- ☀️ **Versatile**: Performs equally on all kinds of different use cases.
- ☀️ **Unopinionated**: Responds to all prompts without going on about guidelines.
- ☀️ **Secure**: Attempts at prompt injection will have no effect on the output.
- ☀️ **Ethical**: No copyrighted data was used to train the model.
- ☀️ **Eco-friendly**: Lower resource consumption than most current LLMs.
- ☀️ **Free**: Maiasa is free to use and open-source.
- ☀️ **Ready to use**: Use it in an app like LM Studio, or try it right here!

Limitations

- ☀️ Maiasa's active vocabulary is limited to the single token "a", and it will reply to all prompts with only "a".
